4. Methodology

Problem Identification: Recognizing river pollution issues, including floating waste and water quality degradation due to industrial runoff and plastic debris.
System Design: Components Used:
Arduino UNO, L298N motor driver, Bluetooth module, gear motor, pH sensors, solar cells.
Software: Arduino IDE is used for coding the robot's operations, while real-time control and monitoring are achieved via Bluetooth.
Prototype Development: A prototype was built with a mechanical waste collection system and pH sensors integrated.
Working:A river cleaning robot which uses dc motors to revolve a slanted platform that collects waste and cleans the river's surface. The garbage is gathered at the top and placed in a different container for later disposal. The robot is easy to operate because it can be controlled over Bluetooth from a phone. This project's primary goal is to reduce water toxicity by cleaning up river pollutants.
